Item

An original Wikidata Q item

new Item(q: string, template: string?, languages: Array, listType: LIST_TYPES)
Parameters
q (string) Wikidata Q value
template (string?) css identifier of a template that is part of the current html page. Must start with an '#'
languages (Array) Comma seperated list of 2 or 3-digit language codes. Default is 'en'. A full list can be found at https://www.wikidata.org/wiki/Help:Wikimedia_language_codes/lists/all
listType (LIST_TYPES = table) A template that is used to seperate properties. If no #template is provided, this will apply to both properties, as well as values within multi-value properties.
